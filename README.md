# Gold-Digger-JavaBegin-RU-

- In this game you need to collect gold, hide from monsters and find a way out of the maze... Good luck!

# Tools and technologies used in this course:

- Java 17+
- SQLite
- Swing

# Patterns:
- Singleton
- Factory Method
- Abstract Factory
- Adapter
- Facade
- Observer
- Strategy
- State
